# MiniSite 
## Project Overview
MiniSite is a clean and responsive single-page website designed for beginners to learn HTML and CSS. It includes a header, hero section, feature boxes, contact form, and footer.It is designed to help students easily learn.

## Seed / Inspiration
This project was created as a first-attempt HTML/CSS mini-assignment. The design focuses on simplicity, readability, and responsive layout using Flexbox and Grid principles.

## Design Rationale
Layout: Single-page structure with semantic HTML (`<header>`, `<section>`, `<footer>`).  
Responsiveness: Works on desktop screens.  
Interactive Elements: Hover effects on buttons and feature boxes.  
Accessibility: Form labels, outlines, and readable color contrast.  

## Features
- Hero section with a button  
- Three feature boxes  
- Contact form with text input, email, textarea, and submit button  
- Footer with copyright  

## Assets
- Images folder: `assets/` 

## How to Run
1. Clone the repository  
2. Open `index.html` in any browser
